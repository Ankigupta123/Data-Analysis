# Airbnb NYC 2019 Data Analysis Project

##  Project Overview

This project analyzes Airbnb New York City listings to uncover pricing behavior, location performance, seasonal demand, and future revenue trends.

The objective was to reverse-engineer stakeholder requirements and transform raw data into actionable business insights using Python, visualization, and time-series forecasting.

Final outputs include an automated executive PDF report containing KPIs, visual analytics, and ARIMA-based forecasting.

---

## Business Objectives

As an Airbnb stakeholder, the following questions were addressed:

- How large is the NYC marketplace?
- What is the average listing price and customer engagement?
- How does pricing vary by neighborhood and room type?
- Is demand seasonal?
- Which locations show higher revenue potential?
- What does future growth look like?
- Can insights be delivered in an executive-friendly format?

## Key Insights

- Manhattan and Brooklyn command premium pricing.
- Entire home listings generate higher revenue.
- Summer months show peak customer activity.
- Forecasting indicates steady revenue growth.

---

## Business Recommendations

- Implement dynamic pricing during peak seasons.
- Prioritize host acquisition in high-value boroughs.
- Expand entire-home inventory in premium neighborhoods.
- Prepare infrastructure for projected growth.
